# 📚 BookCatalog – A Personal Library Management App

A simple and efficient Android app that helps users manage their personal book collection by allowing them to add, edit, delete, and reserve books.

---

## 🚀 Features

- Add, edit, or delete books with title, author, and notes  
- Search for books by name  
- Reserve books for specific members  
- Organized and clean UI with custom fonts and themes

---

## 🛠️ Technologies Used

**Front End**:  

- XML (Layout Design)  
- Kotlin (UI Logic)  
- Android Jetpack Components  

**Back End / Logic**:  
- Kotlin  
- Local Data Storage (List simulation)  
- Repository pattern
